# Model Card: ARPX Risk Stratifier (CatBoost v2.1)

Model card following [Mitchell et al. 2019](https://arxiv.org/abs/1810.03993).

---

## Model details

- **Model name**: ARPX-CatBoost-v2.1
- **Architecture**: Gradient-boosted decision tree ensemble (CatBoost)
  - 500 iterations, depth 6, learning rate 0.05
  - Native categorical handling (no manual one-hot)
- **Output**: Three-class probabilistic classifier
  - Classes: `low`, `moderate`, `high` risk
- **Inputs**: 19 features across 7 domains (see DATA_CARD.md)
- **Auxiliary layers**:
  - Split conformal prediction (LAC nonconformity, q̂=0.7069 at α=0.05)
  - SHAP TreeExplainer (5 top drivers per prediction)
  - Wachter counterfactual optimizer (DE + L-BFGS-B)
- **Training date**: April 2026
- **Author**: Vishal Hota (KL University Hyderabad)

## Intended use

### Primary intended uses
- Methodological research demonstration of uncertainty-aware ML
  for medically underserved populations.
- Educational reference for conformal prediction + counterfactual
  explanations in clinical tabular ML.

### Primary intended users
- ML researchers in clinical decision support.
- Sports medicine clinicians evaluating ML-assisted screening
  tools (with appropriate skepticism).

### Out-of-scope uses
- **Clinical diagnosis**. The model is not a diagnostic instrument.
- **Anti-doping enforcement**. License (CC BY-NC-SA 4.0) explicitly
  prohibits enforcement use.
- **Insurance risk pricing**. License prohibits actuarial use.
- **Self-treatment without clinician oversight**. Frontend gates
  high-risk predictions behind clinician-contact guidance.

## Ethical considerations

### Disclosed in paper Section VII

1. **Label circularity**: Labels generated by a deterministic scoring
   rule. Reported accuracy reflects model approximation of that rule,
   not clinical truth.
2. **Demographic skew**: Synthetic data follows published prevalence
   patterns (predominantly male, Western reference ranges).
3. **Misuse vectors**:
   - Surveillance / anti-doping enforcement → license prohibits
   - Insurance underwriting → license prohibits
   - Self-diagnosis without supervision → UI guidance mitigates
   - Application to non-enhanced athletes → mandatory cycle-phase
     input prevents accidental misuse
4. **Synthetic-data risk**: Model has never seen a real athlete.
   Generalization to real biomarker distributions is unproven.

### Mitigation in progress

- Blind clinical rating study (n=200, ongoing, see Appendix B)

- Public deployment includes honest disclosure footer

## Caveats and recommendations

1. **Use the conformal prediction set, not the point prediction.**
   When the set contains multiple classes, refer for clinical review.
2. **Interpret SHAP drivers as the model's reasoning, not as causal
   evidence.**
3. **Counterfactual changes are heuristic.** A real intervention
   that changes hematocrit will likely also change related markers.
4. **The 92.67% test accuracy is benchmark-recovery accuracy.**
   It is not a clinical-outcome accuracy. The same model on real
   athlete records would likely perform differently.
5. **For populations with documented biomarker variation**
   (e.g., South Asian creatinine, female-specific endocrine ranges),
   the model needs region-specific recalibration before any
   downstream use.
